# 08 - Configuring Docker Agent

This lab keeps Jenkins installation simple. If you want builds to run in Docker, add Docker to the instance or attach a dedicated agent.

## Recommended Learning Path

- do not build on the controller in long-lived environments
- use a dedicated agent or temporary agent container
- prefer isolated build nodes in production

## Local Teaching Shortcut

For a small demo you may install Docker on the instance and run agent workloads there, but call out the tradeoff clearly.

# Learner Workbook

## Before You Begin

- record your AWS region
- record your selected instance type
- record your expected hourly cost

## Checkpoints

1. I can explain why Session Manager is safer than public SSH for this lab.
2. I can identify the EC2 instance ID.
3. I can start a port-forwarding session to Jenkins.
4. I can unlock Jenkins and create the first admin user.
5. I can run a freestyle job.
6. I can run a simple pipeline.
7. I can describe what must be backed up from `JENKINS_HOME`.

## Reflection Questions

- What would break if the EBS volume were deleted?
- Why is storing AWS credentials in Jenkins a poor default for EC2-based access?
- When would a single-instance Jenkins design stop being acceptable?

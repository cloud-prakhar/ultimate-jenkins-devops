# Learner Challenges

## Guided Exercise

1. Run the beginner pipeline.
2. Add a failing test and observe the result.
3. Fix the test and rerun.

## Independent Challenge

1. Add a new endpoint test.
2. Raise the coverage requirement to 90 percent.
3. Add a deployment stage controlled by a parameter.

## Troubleshooting Exercise

- change the agent label from `linux` to `windows`
- run the job and identify why it stays queued
- restore the label and verify recovery

# Module Template

Use this template when creating a new learning module.

## Purpose

The template keeps modules consistent for beginners, instructors, and self-paced learners. It also keeps future maintenance predictable.

## Structure

```text
templates/module-template/
├── README.md
├── theory.md
├── prerequisites.md
├── lab/
│   ├── README.md
│   ├── starter/
│   ├── solution/
│   └── validate.sh
├── exercises/
│   ├── guided-exercise.md
│   ├── independent-challenge.md
│   └── solutions.md
├── troubleshooting.md
├── quiz.md
└── diagrams/
```

## Required Content

- overview
- learning objectives
- prerequisites
- estimated duration
- difficulty
- required environment
- cost information
- theory
- architecture
- lab steps
- expected output
- validation
- break-it exercise
- troubleshooting
- cleanup
- quiz
- independent challenge
- solution
- additional reading
- official references
